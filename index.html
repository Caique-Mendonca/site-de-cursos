O que é Front-end?

Front-end refere-se à parte de um sistema de software ou website que interage diretamente com os usuários finais. Ou seja é a parte visível e interativa do site que os usuários veem e usam diretamente, enquanto o "Back-end" refere-se ao lado do servidor.

O que é Back-end?

Back-end refere-se a parte de um sistema de software que lida com a lógica e processamento de dados. Ou seja é como se fosse a "parte de trás" de um aplicativo ou site, onde o usuário não vê, onde os dados são processados, armazenados e gerenciados.
